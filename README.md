# C2F-DETR
This is the official implementation of the paper "C2F-DETR: A Real-Time Fine-Grained Leaf Disease Detection Model with Progressive Context-to-Frequency Enhancement".
In agricultural scenarios, fine-grained diseased leaf targets are typically small in scale, exhibit subtle inter-class differences, and are highly susceptible to interference from illumination variation, occlusion, and complex backgrounds, posing substantial challenges to existing real-time detection frameworks in terms of insufficient contextual semantic exploitation, degraded detail representation, and unstable query initialization. To address these issues, we propose C2F-DETR, a real-time fine-grained diseased leaf detection model with progressive enhancement from context to the frequency domain. Specifically, a Contextual Semantic Enhancement Mechanism (CSEM) is constructed to aggregate leaf textures and the surrounding contextual information of lesions, thereby enhancing fine-grained discriminative representations. In addition, a High-Low Frequency Modulated Attention (HLFMA) module is developed to jointly model high-frequency boundaries and low-frequency structures, alleviating detail suppression and background interference caused by downsampling. Furthermore, a Center Prior Guided Decoder (CPGD) is designed to stabilize query initialization and iterative updates, thereby improving the localization and detection capability for dense small lesions and weak disease symptoms. Experimental results demonstrate that the proposed method achieves 87.8% mAP on FieldPlant, as well as 49.6% AP and 79.8% mAP on PlantDoc, exhibiting stronger detection performance.
